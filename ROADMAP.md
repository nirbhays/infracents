# InfraCents Roadmap

## Vision
Make cloud cost visible at the exact moment engineers make decisions — before code merges, not after the bill arrives.

## ✅ Shipped
- Real-time Terraform cost estimates on every PR
- AWS and GCP live pricing API integration
- Zero-config GitHub App
- Resource-level cost breakdown in PR comments
- Monthly cost projections

## 🔨 In Progress
- [ ] Azure cost estimation
- [ ] Monthly cost delta vs current state ("this PR costs $47 more/month")
- [ ] Terragrunt support

## 📋 Planned — Q2 2025
- [ ] Cost trend charts across PRs
- [ ] Team spending alerts and budgets
- [ ] Terraform module cost attribution
- [ ] GitLab MR integration

## 📋 Planned — Q3 2025
- [ ] Multi-cloud cost comparison ("same infra on AWS vs GCP vs Azure")
- [ ] Reserved instance / savings plan recommendations
- [ ] FinOps tagging compliance checks
- [ ] Org-level cost dashboard

## 💡 Under Consideration
- Pulumi support
- CDK support
- Slack cost digest reports
